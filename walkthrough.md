# Oreca Medal Web - Play Walkthrough

## 現在の状況 (Current Status)
「モンスター烈伝 オレカバトル2 パンドラのメダル」をモチーフにしたWebブラウザゲームのプロトタイプが完成しました。
以下の機能が実装されています：

- **基本ゲームループ**: 射的ステージ、ボスバトル、リザルト画面
- **操作**: マウス(PC)またはタッチ(スマホ)によるメダル発射
- **システム**:
  - **メダルシステム**: メダルを消費して弾を発射
  - **デッキ選択**: 3体のモンスターからチームを選択（リーダーによって攻撃色とEX技名が変化）
  - **EX召喚**: ゲージMAXで強力な全体攻撃＆ボスへの大ダメージ

## プレイ方法 (How to Play)

### 1. 起動方法 (Setup)
このゲームはESモジュールを使用しているため、ブラウザのセキュリティ制限により`file://`で直接開くと動作しない場合があります。以下のいずれかの方法でローカルサーバーを立ち上げてください。

**推奨: Pythonを使用する場合 (Mac/Linux/Windows)**
ターミナルでプロジェクトのディレクトリ(`/Users/SSS/.gemini/antigravity/brain/46791868-e271-4aaa-aa5e-ec0ad1d797a7`)に移動し、以下のコマンドを実行します：

```bash
python3 -m http.server 8000
```
その後、ブラウザで [http://localhost:8000](http://localhost:8000) にアクセスしてください。

### 2. 操作方法 (Controls)
- **射撃**: 画面の撃ちたい場所をタップ/クリック、またはドラッグ/スワイプし続けると連射します。
- **EX召喚**: 画面右下の「EX GAUGE」が最大まで溜まり、激しく光ったらタップしてください。カットイン演出と共に強力な攻撃が発動します。
